1 Заголовки

Заголовки являются одним из важнейших инструментов для структуирования текста.

1.1 Рекомндации по исполльованию

Вот некоторые рекомендации по исполльованию заголовков в HTML - странице.

1.1.1 Использоваие H1

Не рекомендуется, чтобы на одной странице содержалось несколько заголовков верзнего уровня.

1.1.2 ИспользоваиеH2-H6

При использовании подзаголовков не рекомендуется пропускать уровни заголовков, то есть после заголовка H1 должен идти подзаголовок H2 и только потом подзаоловок H3.
